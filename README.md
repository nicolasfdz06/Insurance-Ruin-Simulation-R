# Insurance Ruin Simulation (R)

## Overview
This repository contains a group university coursework project involving the use of R to model an insurer’s fund and estimate ruin probabilities under different assumptions.

## Project Objective
The aim of the project was to analyse the probability that an insurer’s fund becomes negative over a 10-year horizon. The model considered quarterly asset income and losses, and examined how ruin probabilities changed under different dependence structures and economic conditions.

## What the Model Does
The project models the insurer’s fund over 40 quarters and estimates ruin probabilities using:
- a normal approximation benchmark
- Monte Carlo simulation
- correlated loss modelling
- a Markov-chain extension for economic states

## Key Features
- Normal approximation for baseline ruin probability
- Monte Carlo estimation of ruin probabilities
- Simulation of fund paths over time
- Sensitivity analysis with varying correlation assumptions
- Markov-state economic model with stationary distribution analysis
- R code for simulation, estimation, and visualisation
